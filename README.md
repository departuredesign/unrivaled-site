# Unrivaled Sports — Website

Static marketing site for [UnrivaledSports.com](https://unrivaledsports.com).

## Pages

- `index.html` — Homepage (hero, about, brands, press, Instagram, footer)
- `team.html` — Leadership team page

## Deployment

Deployed via [Vercel](https://vercel.com). Push to `main` to deploy.

### Local Preview

Open `index.html` in a browser, or use any static server:

```bash
npx serve .
```

## Notes

- Self-contained HTML files (CSS + JS inline) — no build step required
- Custom fonts (Review, Neue Plak) will need to be added as web font files
- Placeholder images (`ph` class elements) are wireframe boxes — replace with real assets
- Future migration path: WordPress
